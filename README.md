# Run this project
1. Install http://nodejs.org/ .
2. Install npm bower
    ```bash
      $  npm install -g bower
    ```
3. Get dependency
    ```bash
      $   bower install
    ```


# Build this project.
1. Install gulp and gulp plugins
    ```bash
        npm install gulp
        npm install gulp-uglify
        npm install gulp-concat
    ```
2. Run gulp
    ```bash
        gulp
    ```
